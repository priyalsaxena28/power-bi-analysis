# 📊 TechyStore Sales Analysis Dashboard (Power BI)

## Overview

The **TechyStore Sales Analysis Dashboard** is an interactive Power BI project designed to analyze sales performance across products, categories, customers, regions, and time periods. The dashboard transforms raw sales data into meaningful business insights, helping stakeholders monitor KPIs, identify trends, and support data-driven decision-making.

This project demonstrates practical skills in **Power BI, DAX, Power Query, data modeling, and data visualization**.

## Objectives

* Analyze overall sales performance.
* Track revenue, profit, quantity sold, and customer trends.
* Compare sales across different product categories.
* Identify top-performing products and regions.
* Monitor monthly and yearly sales trends.
* Build an interactive dashboard using slicers and filters.

## Features

* Interactive KPI Cards
* Sales Trend Analysis
* Product Category Analysis
* Regional Sales Performance
* Customer Insights
* Top & Bottom Performing Products
* Dynamic Filters and Slicers
* Time-Based Analysis using Calendar Table
